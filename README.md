# 🚀 CareerPilot AI

![Python](https://img.shields.io/badge/Python-3.13-blue)
![Flask](https://img.shields.io/badge/Flask-Web%20Framework-black)
![SQLite](https://img.shields.io/badge/Database-SQLite-blue)
![Gemini AI](https://img.shields.io/badge/AI-Gemini-purple)
![License](https://img.shields.io/badge/License-MIT-green)

CareerPilot AI is a modern AI-powered Resume Analyzer and Job Match Platform that helps job seekers optimize their resumes for Applicant Tracking Systems (ATS), compare resumes against job descriptions, and receive AI-powered recommendations to improve their chances of getting shortlisted.

---

# ✨ Key Features

### 🔐 Secure Authentication

* User registration and login
* Secure password hashing
* Session management

### 📄 Resume Upload

* Upload PDF and DOCX resumes
* Automatic resume storage
* Resume history tracking

### 📝 Resume Parsing

Automatically extracts:

* Name
* Email
* Phone Number
* Skills
* Education
* Experience
* Projects

### 📊 ATS Compatibility Analysis

* ATS score (0–100)
* Resume strengths
* Missing sections
* Resume weaknesses
* Keyword analysis

### 🎯 Job Description Matching

* Compare resumes with job descriptions
* Skill match percentage
* Missing skills
* Keyword coverage

### 🤖 AI Resume Suggestions

Powered by Google Gemini AI to generate:

* Resume improvement suggestions
* Better professional summaries
* Project enhancement ideas
* ATS optimization tips

### 📈 Interactive Dashboard

* Resume statistics
* ATS analytics
* Analysis history
* Visual score cards
